# App Assets

This directory should contain the following assets for the Expo app:

- `icon.png` - App icon (1024x1024px)
- `splash.png` - Splash screen (1242x2436px recommended)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px)

You can generate these assets using:
- Run `yarn generate-assets` (or `npm run generate-assets`) to create placeholder assets
- [Expo Asset Generator](https://www.npmjs.com/package/@expo/asset-generator) - `yarn add -D @expo/asset-generator`
- [App Icon Generator](https://www.appicon.co/)
- Or create them manually

The placeholder assets are already generated and ready to use. Replace them with your custom designs when ready.
